# Outperform

**FunnelEnvy's hiring challenges for AI-native professionals.**

FunnelEnvy is a B2B conversion optimization agency building AI-powered agents that learn client businesses and compound intelligence over time.

We don't hire from resumes. We hire from work.

---

## How It Works

1. Pick a challenge from the table below
2. Read the brief
3. Fork this repo
4. Do the work
5. Open a PR with your submission
6. We blind-review your work against an AI-generated baseline
7. If you outperform the AI, we want to talk

## Why This Exists

AI can generate competent work. We need people who can generate exceptional work - the kind that requires taste, judgment, and strategic thinking that AI can't replicate on its own.

If your submission ties with Claude's output, that tells us you're competing with a tool. We're looking for people who make the tool look like a first draft.

## Submission Process

### 1. Fork this repo

### 2. Create a branch

```
git checkout -b submission/firstname-lastname
```

### 3. Add your work

Place your submission files in the challenge's submissions directory:

```
challenges/[challenge-name]/submissions/firstname-lastname/
```

### 4. Open a PR

Open a pull request back to this repo targeting `main`. Use the submission PR template - it will populate automatically.

### 5. Wait for review

Submissions are anonymized and blind-reviewed against the AI baseline. We'll respond within 5 business days.

---

## The Rules

- Use whatever tools you want. AI, plugins, libraries, collaborators, whatever.
- The baseline was generated by Claude using the same brief you're seeing. If you just paste the brief into an AI, you'll get roughly the same answer we already have. That's a tie. **Ties don't advance.**
- To outperform, you need to add something AI can't: creative judgment, strategic instinct, domain expertise, or craft that goes beyond competent.
- Each challenge lists a time target. We're testing quality of thinking, not endurance.
